# 油漆店官网 Skill

## 项目概述
为油漆个体工商户制作一个**静态展示官网**，用于线上展示商品和价格区间，不涉及在线支付。网站将部署到 GitHub Pages 并绑定自定义域名。

## 核心需求

### 1. 前台展示页面

#### 首页
- 大图轮播展示（支持图片和视频）
- 店铺简介
- 热门/推荐商品展示
- 快速导航到各商品分类

#### 商品展示页
- 按分类展示油漆产品（如：内墙漆、外墙漆、木器漆、底漆等）
- 每个商品卡片包含：
  - 商品图片（支持多图轮播）
  - 商品视频（产品效果展示）
  - **价格区间**（如 ¥200-500/桶）
  - 简要描述
- 点击进入商品详情页：
  - 大图/视频展示
  - 详细规格参数
  - 适用场景说明
  - 使用方法

#### 关于我们
- 店铺历史和介绍
- 经营优势（正品保障、专业服务等）
- 资质证书展示

#### 联系方式
- 店铺地址
- 联系电话
- 微信二维码
- 营业时间
- 地图定位（可选）

### 2. 后台管理页面

#### 核心功能
- **商品管理**：添加、编辑、删除商品
- **上下架控制**：一键切换商品显示状态
- **媒体管理**：上传和管理图片、视频
- **分类管理**：管理商品分类
- **数据导出**：导出商品数据备份

#### 技术实现方案
- 纯前端实现，无需服务器
- 使用 **localStorage** 存储管理数据
- 使用 **GitHub API** 同步数据到仓库（需配置 Personal Access Token）
- 或使用 **JSON 文件** 直接管理商品数据

### 3. 技术栈
- **前端框架**：HTML5 + Tailwind CSS + 原生 JavaScript
- **构建工具**：无（纯静态）
- **部署平台**：GitHub Pages
- **数据存储**：JSON 文件 + localStorage 缓存

## 设计规范
- **风格**：高端简约，参考国际涂料品牌（Benjamin Moore、Sherwin-Williams）
- **色彩**：以油漆色卡为灵感，主色调待定
- **响应式**：移动端优先，适配手机、平板、桌面
- **性能**：图片懒加载，视频按需加载

## 部署计划
- **代码仓库**：paint-store（当前仓库）
- **部署方式**：GitHub Pages（从 main 分支部署）
- **自定义域名**：待用户提供，将配置 CNAME 文件
- **HTTPS**：GitHub Pages 自动支持
